# 配置 OpenAI 环境变量

> 注意：如果你已经通过 AGICTO 获得了免费的 API Key，请继续以下步骤。

## 步骤 1: 创建 `.env` 文件

在当前项目文件夹中，创建一个名为 `.env` 的文件，文件内容如下：

<CodeGroup>
```bash {language: 'bash'}
OPENAI_API_KEY=sk-xxx
OPENAI_BASE_URL=https://api.agicto.cn/v1
OPENAI_API_BASE="https://api.agicto.cn/v1"
```
</CodeGroup>

## 步骤 2: 为 LLamaIndex 配置 OpenAI API Base

在 LlamaIndex 中集成 AGICTO 需要进行额外的配置。你可以直接在 `.env` 文件中添加以下内容，或者在代码中显式设置。

<CodeGroup>
```bash {language: 'bash'}
OPENAI_API_BASE="https://api.agicto.cn/v1"
```
</CodeGroup>

## 步骤 3: 在代码中加载环境变量

为了使代码可以读取 `.env` 文件中的配置，你需要在代码文件中增加以下内容：

<CodeGroup>
```python {language: 'python'}
# 加载 .env 文件到环境变量
from dotenv import load_dotenv, find_dotenv

# 查找并加载 .env 文件
_ = load_dotenv(find_dotenv())
```
</CodeGroup>

通过这些步骤，你的 LlamaIndex 项目现在已经正确配置好，可以使用 AGICTO 提供的 API 进行操作。确保 API Key 安全保存，并且不要公开分享 `.env` 文件。
