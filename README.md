# Agicto Documentation Content

This repository contains all documentation content for the Agicto platform.

## Structure

- `docs/` - Documentation pages in MDX format
- `api-reference/` - API reference documentation

## Content Format

All documentation files are written in MDX (Markdown + JSX) format.

### Frontmatter

Each MDX file should include frontmatter with metadata:

```yaml
---
title: Page Title
description: Page description
slug: page-slug
---
```

## Contributing

1. Create or edit MDX files in the appropriate directory
2. Commit changes with descriptive messages
3. Push to the main branch

The documentation site will automatically rebuild when changes are pushed.
