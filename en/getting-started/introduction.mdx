# Introduction to Agicto

Agicto is a unified AI model access platform that lets you reach 1000+ large language models worldwide through a single API.

## Why Choose Agicto?

### Fully Compatible with the OpenAI SDK

You do not need to learn a new SDK or API. Just keep using the OpenAI SDK you already know:

```python
from openai import OpenAI

# Only change these two parameters
client = OpenAI(
    api_key="your-api-key",        # Use your Agicto API key
    base_url="https://api.agicto.cn/v1"  # Use the Agicto base URL
)

# Everything else stays the same
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### Switch Models by Changing Only the Model Name

With the same code, you can switch to any supported model by changing only the `model` parameter:

```python
# Use GPT-4o-mini
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)

# Switch to Claude
response = client.chat.completions.create(
    model="claude-3.5-sonnet",
    messages=[{"role": "user", "content": "Hello!"}]
)

# Switch to Qwen
response = client.chat.completions.create(
    model="qwen-max",
    messages=[{"role": "user", "content": "Hello!"}]
)

# Switch to DeepSeek
response = client.chat.completions.create(
    model="deepseek-v3",
    messages=[{"role": "user", "content": "Hello!"}]
)
```
