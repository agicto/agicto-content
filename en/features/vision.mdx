# Vision Models

Vision models can analyze images for description, OCR, visual question answering, chart analysis, and other multimodal tasks.

## Basic usage

### Analyze an image URL

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "text", "text": "What is in this image?"},
                {
                    "type": "image_url",
                    "image_url": {
                        "url": "https://example.com/image.jpg"
                    }
                }
            ]
        }
    ]
)

print(response.choices[0].message.content)
```

### Analyze a local image

```python
import base64

def encode_image(image_path):
    with open(image_path, "rb") as image_file:
        return base64.b64encode(image_file.read()).decode("utf-8")

image_base64 = encode_image("path/to/image.jpg")

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "text", "text": "Describe this image"},
                {
                    "type": "image_url",
                    "image_url": {
                        "url": f"data:image/jpeg;base64,{image_base64}"
                    }
                }
            ]
        }
    ]
)
```

```python
        }
    ]
```


## Common scenarios

- Image description
- OCR and text extraction
- Visual question answering
- Chart interpretation
- Code screenshot explanation

## Recommended models

- `gpt-4o`
- `gpt-4.1`
- `gemini-2.0-flash`
- `qwen-vl-max`

Choose the model based on latency, image complexity, and whether you need strong multilingual support.
