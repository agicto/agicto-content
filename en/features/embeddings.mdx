# Embeddings

Embeddings convert text into numeric vectors for semantic search, retrieval, clustering, and recommendation systems.

## What are embeddings

Embeddings map text into fixed-length vectors. Semantically similar texts end up with similar vector representations.

**Common use cases**
- Semantic search and similarity
- Retrieval and question answering
- Text classification and clustering
- Recommendation systems

## Basic usage

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.embeddings.create(
    model="text-embedding-3-small",
    input="Artificial intelligence is changing the world"
)

embedding = response.data[0].embedding
print(f"Vector length: {len(embedding)}")
print(f"First 5 values: {embedding[:5]}")
```

## Batch processing

```python
texts = [
    "Artificial intelligence is changing the world",
    "Machine learning is a core AI technology",
    "Deep learning accelerated modern AI"
]

response = client.embeddings.create(
    model="text-embedding-3-small",
    input=texts
)

embeddings = [data.embedding for data in response.data]
print(f"Processed {len(embeddings)} texts")
```

## Similarity calculation

```python
import numpy as np

def cosine_similarity(a, b):
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

response1 = client.embeddings.create(
    model="text-embedding-3-small",
    input="artificial intelligence"
)
response2 = client.embeddings.create(
    model="text-embedding-3-small",
    input="machine learning"
)

emb1 = response1.data[0].embedding
emb2 = response2.data[0].embedding

similarity = cosine_similarity(emb1, emb2)
print(f"Similarity: {similarity}")
```

## Semantic search example

```python
documents = [
    "Python is a high-level programming language",
    "Machine learning is a branch of artificial intelligence",
    "Deep learning uses neural networks",
    "Natural language processing works with text data"
]

doc_response = client.embeddings.create(
    model="text-embedding-3-small",
    input=documents
)
doc_embeddings = [data.embedding for data in doc_response.data]

query = "What is AI?"
query_response = client.embeddings.create(
    model="text-embedding-3-small",
    input=query
)
query_embedding = query_response.data[0].embedding

similarities = [
    cosine_similarity(query_embedding, doc_emb)
    for doc_emb in doc_embeddings
]
```

## Supported embedding models

### OpenAI

| Model | Dimensions | Notes |
|------|------|------|
| text-embedding-3-small | 1536 | Small, cost-effective |
| text-embedding-3-large | 3072 | Best quality |
| text-embedding-ada-002 | 1536 | Previous-generation model |

### Cohere

| Model | Dimensions | Notes |
|------|------|------|
| embed-english-v3.0 | 1024 | English embedding model |
| embed-multilingual-v3.0 | 1024 | Multilingual model |
| embed-english-light-v3.0 | 384 | Lightweight English model |

### Voyage AI

| Model | Dimensions | Notes |
|------|------|------|
| voyage-large-2 | 1536 | Large model |
| voyage-code-2 | 1536 | Code embeddings |
| voyage-2 | 1024 | General-purpose model |

### Jina AI

| Model | Dimensions | Notes |
|------|------|------|
| jina-embeddings-v2-base-en | 768 | English base model |
| jina-embeddings-v2-base-zh | 768 | Chinese base model |
| jina-embeddings-v2-base-code | 768 | Code embedding model |

## Selection guidance

- **English tasks**: `text-embedding-3-large`, `voyage-large-2`
- **Balanced cost/performance**: `text-embedding-3-small`, `embed-english-v3.0`
- **Chinese tasks**: `text-embedding-3-large`, `jina-embeddings-v2-base-zh`
- **Multilingual tasks**: `embed-multilingual-v3.0`
- **Code tasks**: `voyage-code-2`, `jina-embeddings-v2-base-code`

## Best practices

1. Batch multiple texts in one request.
2. Cache vectors whenever possible.
3. Pick the right dimension for the task.
4. Normalize vectors before similarity search.
5. Chunk long texts before embedding.
