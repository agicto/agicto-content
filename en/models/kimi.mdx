# Kimi Models

The Kimi family of large language models provided by Moonshot AI, designed for long-context workloads.

## Main models

| Model | Context | Notes |
|------|--------|------|
| kimi-k1.5 | - | Latest Kimi reasoning model |
| kimi-k2-thinking | - | Chain-of-thought reasoning model |
| moonshot-v1-128k | 128K | 128K context version |
| moonshot-v1-1m | 1M | Ultra-long-context version |

## Highlights

- **Long context**: Supports up to 1M tokens
- **Chinese optimization**: Strong performance on Chinese tasks
- **Document workflows**: Well suited for long documents and long conversations

## Example

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.chat.completions.create(
    model="moonshot-v1-128k",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```
