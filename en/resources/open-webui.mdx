# Open WebUI Integration

Open WebUI is an open-source ChatGPT-style interface that supports multiple LLM backends.

## Configure Agicto

### 1. Open settings

Click the avatar in the top-right corner, then go to **Settings → Connections**.

### 2. Configure the OpenAI API

- **API Base URL**: `https://api.agicto.cn/v1`
- **API Key**: your Agicto API key

### 3. Save the configuration

After saving, Open WebUI will load the available model list automatically.

## Common features

### Multi-model chat

- Switch between different models in the chat UI
- Compare GPT-4o, Claude, Gemini, and others
- Test output quality across providers

### Document upload

- Upload PDF, Word, TXT, and other files
- Ask questions against uploaded content
- Support multi-document retrieval

### Vision

- Upload images for analysis
- Use vision-capable models like `gpt-4o` and `claude-3.5-sonnet`
- Perform image description and OCR tasks

### Voice chat

- Voice input
- Text-to-speech output
- Realtime voice interaction

## Recommended models

- **Daily chat**: `gpt-4o-mini`, `deepseek-v3`
- **Complex tasks**: `gpt-4o`, `claude-3.5-sonnet`
- **Chinese conversation**: `qwen-max`, `ernie-4.0`
- **Code generation**: `deepseek-coder`, `gpt-4o`

See [Model Overview](/models/models) for the full model list.
