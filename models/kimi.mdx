# Kimi 模型

月之暗面（Moonshot AI）提供的 Kimi 系列大语言模型，支持超长上下文。

## 主要模型

| 模型 | 上下文 | 说明 |
|------|--------|------|
| moonshot-v1-8k | 8K | Kimi 8K 版本 |
| moonshot-v1-32k | 32K | Kimi 32K 版本 |
| moonshot-v1-128k | 128K | Kimi 128K 版本 |
| moonshot-v1-1m | 1M | Kimi 超长上下文版本 |

## 特点

- **超长上下文**: 支持最长 1M token 上下文
- **中文优化**: 针对中文场景优化
- **长文档处理**: 适合处理长文档、长对话

## 使用示例

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.chat.completions.create(
    model="moonshot-v1-128k",
    messages=[{"role": "user", "content": "你好！"}]
)

print(response.choices[0].message.content)
```
