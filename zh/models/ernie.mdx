# 文心一言模型

百度提供的文心一言（ERNIE）系列大语言模型。

## 主要模型

| 模型 | 上下文 | 说明 |
|------|--------|------|
| ernie-4.0 | 8K | 文心一言 4.0 版本 |
| ernie-4.5-vl | 30K | 文心一言 4.5 视觉版本 |
| ernie-bot-turbo | 8K | 文心一言 Turbo 快速版 |
| ernie-speed | 128K | 文心一言 Speed 版本 |
| ernie-lite | 8K | 文心一言 Lite 轻量版 |
| ernie-tiny | 8K | 文心一言 Tiny 极速版 |

## 使用示例

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.chat.completions.create(
    model="ernie-4.0",
    messages=[{"role": "user", "content": "你好！"}]
)

print(response.choices[0].message.content)
```
