# Open WebUI 集成

Open WebUI 是一个开源的 ChatGPT 风格界面，支持多种 LLM 后端。

## 配置 Agicto

### 1. 进入设置

点击右上角头像 → **设置 → 连接**

### 2. 配置 OpenAI API

- **API Base URL**: `https://api.agicto.cn/v1`
- **API Key**: 输入你的 Agicto API Key

### 3. 保存配置

点击保存，系统会自动加载可用模型列表

## 使用功能

### 多模型对话

- 在对话界面选择不同的模型
- 支持切换 GPT-4o、Claude、Gemini 等
- 可以对比不同模型的回答

### 文档上传

- 上传 PDF、Word、TXT 等文档
- 基于文档内容进行问答
- 支持多文档联合检索

### 图像理解

- 上传图片进行分析
- 支持 GPT-4o、Claude 3.5 等视觉模型
- 可以进行图片描述、OCR 等任务

### 语音对话

- 支持语音输入
- 支持语音输出（TTS）
- 实时语音对话

## 推荐模型

- **日常对话**: gpt-4o-mini, deepseek-v3
- **复杂任务**: gpt-4o, claude-3.5-sonnet
- **中文对话**: qwen-max, ernie-4.0
- **代码生成**: deepseek-coder, gpt-4o

查看 [模型总览](/models/models) 了解所有可用模型。
