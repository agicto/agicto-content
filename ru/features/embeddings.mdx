# Эмбеддинги

Эмбеддинги преобразуют текст в числовые векторы для семантического поиска, retrieval, кластеризации и рекомендаций.

## Что такое эмбеддинги

Эмбеддинги переводят текст в векторы фиксированной длины. Семантически близкие тексты дают похожие векторные представления.

**Типичные сценарии**
- Семантический поиск и similarity
- Retrieval и question answering
- Классификация и кластеризация текста
- Рекомендательные системы

## Базовое использование

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.embeddings.create(
    model="text-embedding-3-small",
    input="Artificial intelligence is changing the world"
)

embedding = response.data[0].embedding
print(f"Длина вектора: {len(embedding)}")
print(f"Первые 5 значений: {embedding[:5]}")
```

## Пакетная обработка

```python
texts = [
    "Artificial intelligence is changing the world",
    "Machine learning is a core AI technology",
    "Deep learning accelerated modern AI"
]

response = client.embeddings.create(
    model="text-embedding-3-small",
    input=texts
)
```

## Вычисление similarity

```python
import numpy as np

def cosine_similarity(a, b):
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))
```

## Поддерживаемые модели

### OpenAI

| Модель | Размерность | Описание |
|------|------|------|
| text-embedding-3-small | 1536 | Компактная и выгодная |
| text-embedding-3-large | 3072 | Лучшее качество |
| text-embedding-ada-002 | 1536 | Модель предыдущего поколения |

### Cohere

| Модель | Размерность | Описание |
|------|------|------|
| embed-english-v3.0 | 1024 | Англоязычная модель |
| embed-multilingual-v3.0 | 1024 | Мультиязычная модель |
| embed-english-light-v3.0 | 384 | Облегченная англоязычная модель |

### Voyage AI

| Модель | Размерность | Описание |
|------|------|------|
| voyage-large-2 | 1536 | Большая модель |
| voyage-code-2 | 1536 | Эмбеддинги для кода |
| voyage-2 | 1024 | Универсальная модель |

### Jina AI

| Модель | Размерность | Описание |
|------|------|------|
| jina-embeddings-v2-base-en | 768 | Базовая английская модель |
| jina-embeddings-v2-base-zh | 768 | Базовая китайская модель |
| jina-embeddings-v2-base-code | 768 | Эмбеддинги для кода |

## Рекомендации

- **Английские задачи**: `text-embedding-3-large`, `voyage-large-2`
- **Баланс цена/качество**: `text-embedding-3-small`, `embed-english-v3.0`
- **Китайские задачи**: `text-embedding-3-large`, `jina-embeddings-v2-base-zh`
- **Мультиязычные задачи**: `embed-multilingual-v3.0`
- **Код**: `voyage-code-2`, `jina-embeddings-v2-base-code`

## Лучшие практики

1. Обрабатывайте несколько текстов за один запрос.
2. Кэшируйте готовые векторы.
3. Подбирайте размерность под задачу.
4. Нормализуйте векторы перед similarity search.
5. Разбивайте длинные тексты на чанки.
