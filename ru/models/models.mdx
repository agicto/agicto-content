# Обзор моделей

Agicto дает доступ к 1000+ AI-моделям через единый API.

Полный каталог моделей и актуальные цены доступны на [https://agicto.com/model](https://agicto.com/model).

## Ведущие мировые модели

### OpenAI

OpenAI предоставляет сильные модели для чата, reasoning, изображений, аудио и эмбеддингов.

**Основные модели**
- **gpt-5**: лучший выбор для кодинга и agent-задач
- **gpt-4.1**: сильная универсальная non-reasoning модель
- **o3**: продвинутая reasoning-модель
- **gpt-4o-mini**: быстрая и экономичная

**Возможности**
- Генерация изображений
- Распознавание речи
- Эмбеддинги
- Realtime-аудио

[Подробнее](/models/openai)

### Claude (Anthropic)

Claude известен длинным контекстом, сильным reasoning и качественной генерацией текста.

**Основные модели**
- **claude-opus-4**
- **claude-sonnet-4-5**
- **claude-haiku-4-5**

**Возможности**
- До 200K контекста

- Надежная длинная генерация

[Подробнее](/models/claude)

[Подробнее](/models/gemini)

DeepSeek предлагает удобные open-source модели с сильными возможностями для кода.

## Как выбрать модель

### По типу задачи

- **Сложное reasoning**: o3, claude-opus-4, kimi-k1.5
- **Повседневный чат**: gpt-4o-mini, deepseek-v3, qwen-turbo
- **Кодинг**: gpt-5, claude-sonnet-4-5, deepseek-v3.2
- **Китайские задачи**: qwen-max, ernie-4.0, hunyuan-large
- **Длинные документы**: moonshot-v1-1m, claude-opus-4, hunyuan-lite
- **Мультимодальность**: gpt-4o, gemini-2.5-flash, qwen-vl-max

### По соотношению цена/качество

- **Бюджетные**: gpt-4o-mini, deepseek-v3, doubao-lite
- **Сбалансированные**: qwen-plus, hunyuan-turbo, gemini-1.5-flash
- **Максимальные возможности**: gpt-5, claude-opus-4, kimi-k1.5

## Пример

Все модели используют один и тот же API. Меняется только параметр `model`:

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)

response = client.chat.completions.create(
    model="claude-sonnet-4-5",
    messages=[{"role": "user", "content": "Hello!"}]
)

response = client.chat.completions.create(
    model="qwen-max",
    messages=[{"role": "user", "content": "Hello!"}]
)
```
