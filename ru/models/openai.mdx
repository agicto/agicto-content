# Модели OpenAI

Большие языковые модели OpenAI, включая GPT-5, GPT-4.1 и reasoning-модели серии O.

## Передовые модели

|------|------|
| gpt-5 | Самая мощная модель для кодинга и agent-задач |
| gpt-5-mini | Более быстрая и экономичная GPT-5 |
| gpt-5-nano | Самая быстрая и дешевая GPT-5 |
| gpt-5-pro | Профессиональная версия GPT-5 |

## Пример

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```
