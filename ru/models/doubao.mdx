# Модели Doubao

Семейство больших языковых моделей Doubao от ByteDance.

## Основные модели

| Модель | Описание |
|------|------|
| doubao-pro | Профессиональная версия |
| doubao-lite | Облегченная версия |
| doubao-character | Версия для ролевого общения |

## Пример

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.chat.completions.create(
    model="doubao-pro",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```
