# Модели ERNIE

Семейство больших языковых моделей ERNIE от Baidu.

## Основные модели

| Модель | Контекст | Описание |
|------|--------|------|
| ernie-4.0 | 8K | ERNIE 4.0 |
| ernie-4.5-vl | 30K | Визуальная модель ERNIE 4.5 |
| ernie-bot-turbo | 8K | ERNIE Turbo |
| ernie-speed | 128K | ERNIE Speed |
| ernie-lite | 8K | ERNIE Lite |
| ernie-tiny | 8K | ERNIE Tiny |

## Пример

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.agicto.cn/v1"
)

response = client.chat.completions.create(
    model="ernie-4.0",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```
