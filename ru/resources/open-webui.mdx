# Интеграция с Open WebUI

Open WebUI — это open-source интерфейс в стиле ChatGPT с поддержкой разных LLM-бэкендов.

## Настройка Agicto

### 1. Откройте настройки

Нажмите на аватар в правом верхнем углу и перейдите в **Settings → Connections**.

### 2. Настройте OpenAI API

- **API Base URL**: `https://api.agicto.cn/v1`
- **API Key**: ваш Agicto API key

### 3. Сохраните конфигурацию

После сохранения Open WebUI автоматически загрузит список доступных моделей.

## Основные возможности

### Мультимодельный чат

- Переключение между разными моделями в одном интерфейсе
- Сравнение GPT-4o, Claude, Gemini и других
- Проверка качества ответов разных провайдеров

### Загрузка документов

- Загрузка PDF, Word, TXT и других файлов
- Вопросы по содержимому документов
- Поддержка retrieval по нескольким документам

### Работа с изображениями

- Загрузка изображений для анализа
- Использование vision-моделей вроде `gpt-4o` и `claude-3.5-sonnet`
- Описание изображений и OCR

### Голосовой чат

- Голосовой ввод
- Text-to-speech
- Realtime голосовое взаимодействие

## Рекомендуемые модели

- **Повседневный чат**: `gpt-4o-mini`, `deepseek-v3`
- **Сложные задачи**: `gpt-4o`, `claude-3.5-sonnet`
- **Китайский диалог**: `qwen-max`, `ernie-4.0`
- **Генерация кода**: `deepseek-coder`, `gpt-4o`

Полный список моделей есть в [обзоре моделей](/models/models).
