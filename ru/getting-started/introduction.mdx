# Введение в Agicto

Agicto — это единая платформа доступа к AI-моделям, которая позволяет работать с более чем 1000 большими языковыми моделями по всему миру через один API.

## Почему стоит выбрать Agicto?

### Полная совместимость с OpenAI SDK

Вам не нужно изучать новый SDK или API. Просто продолжайте использовать знакомый вам OpenAI SDK:

```python
from openai import OpenAI

# Нужно изменить только эти два параметра
client = OpenAI(
    api_key="your-api-key",        # Используйте API key Agicto
    base_url="https://api.agicto.cn/v1"  # Используйте Base URL Agicto
)

# Весь остальной код остается тем же
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### Переключение моделей заменой только имени модели

С тем же самым кодом вы можете переключаться между моделями, меняя только параметр `model`:

```python
# Использовать GPT-4o-mini
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)

# Переключиться на Claude
response = client.chat.completions.create(
    model="claude-3.5-sonnet",
    messages=[{"role": "user", "content": "Hello!"}]
)

# Переключиться на Qwen
response = client.chat.completions.create(
    model="qwen-max",
    messages=[{"role": "user", "content": "Hello!"}]
)

# Переключиться на DeepSeek
response = client.chat.completions.create(
    model="deepseek-v3",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### Один API для всех моделей

С помощью Agicto вы получаете доступ к:

- **Ведущим мировым моделям**: GPT-4o, Claude 3.5, Gemini 2.0, DeepSeek V3
- **Китайским моделям**: Qwen, ERNIE, Hunyuan, Doubao
- **Open-source моделям**: Llama, Mistral и другим

Вам не нужны несколько аккаунтов на разных платформах и несколько API-ключей.

## Поддерживаемые модели
